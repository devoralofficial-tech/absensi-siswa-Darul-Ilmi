# Project Overview

- **Purpose**: Modern student attendance management system (Absensi Siswa) for Darul Ilmi, with features like QR Code scanning for check-ins/check-outs.
- **Tech Stack**:
  - Laravel Framework (PHP 8.3+)
  - Blade + Vanilla JS + CSS / TailwindCSS (Frontend)
  - Vite (Frontend asset bundler)
  - MySQL (Database)
  - Breeze (Authentication starter kit)
- **Directory Structure**:
  - `app/`: Application core code (Models, Controllers, Services).
  - `config/`: Configuration files.
  - `database/`: Database migrations, seeders, and factories.
  - `resources/`: Blade views and assets (JS, CSS).
  - `routes/`: Web and API routing rules.
  - `public/`: Publicly accessible static assets.
